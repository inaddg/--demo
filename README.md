# 练习项目-Vue3 大事件管理系统（记录）
一个基于 Vue3 生态的现代化后台数据管理系统，用于文章与分类的全流程管理。

## 🌐 在线体验
- **在线演示**：[https://fe-bigevent-web.itheima.net/login](https://fe-bigevent-web.itheima.net/login)
- **接口文档**：[https://apifox.com/apidoc/shared-26c67aee-0233-4d23-aab7-08448fdf95ff/api-93850835](https://apifox.com/apidoc/shared-26c67aee-0233-4d23-aab7-08448fdf95ff/api-93850835)
- **接口根路径**：[http://big-event-vue-api-t.itheima.net](http://big-event-vue-api-t.itheima.net)

## 🛠️ 技术栈
- **核心框架**：Vue3 (Composition API)
- **状态管理**：Pinia (含持久化处理)
- **路由管理**：Vue Router 4
- **构建工具**：Vite
- **HTTP 请求**：Axios
- **UI 组件库**：Element Plus (表单校验、表格处理、组件封装)
- **包管理**：pnpm
- **代码规范**：ESLint + Prettier
- **Git 钩子**：husky (代码提交前校验)
- **基础语法**：ES6+

## ✨ 核心功能
### 1. 身份认证
- 用户登录 / 注册
- 记住我、找回密码
- 表单数据校验

### 2. 文章分类管理
- 分类列表展示
- 新增 / 编辑 / 删除分类
- 分类别名管理

### 3. 文章管理
- 文章筛选（按分类、发布状态）
- 文章发布 / 编辑 / 删除
- 分页查询
- 发布状态管理

### 4. 个人中心
- 个人信息管理
- 密码修改等操作

## 🚀 开发亮点
- 采用 **Vue3 Composition API** 开发，代码复用性更强
- **Pinia 持久化** 处理，保证页面刷新后状态不丢失
- **Element Plus** 深度封装，高效实现表单与表格场景
- **pnpm 包管理** 提升安装速度与磁盘空间利用率
- 集成 **ESLint + Prettier + husky**，保障代码规范与提交质量
- 支持 **AI 大模型辅助开发**，提升开发效率
